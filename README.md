# Econometric and Regression Analysis for the US Presidential Election 

## Overview

This project investigates the most influential predictors of U.S. presidential election outcomes using econometric modeling techniques in R. We analyze state-level data from the 2024 US election and evaluate the predictive power of the money raised by Democratic and Republican, unemployment rate, crime rate, education rate, and population.

## Methodology

- Simple Linear Regression (SLR)
- Multiple Linear Regression (MLR)
- Logistic Regression
- Variance Inflation Factor (VIF) diagnostics
- Breusch-Pagan test (heteroskedasticity)
- ROC Curve & AUC evaluation
- Confusion Matrix analysis

## Key Findings

- Campaign fundraising is the strongest predictor of electoral success for both parties
- Logistic regression using fundraising share achieved approximately 96% classification accuracy with an AUC of 0.993
- Other predictors showed limited or inconsistent statistical significance

## Usage
1. Clone the repository
2. Install required R packages
3. Run scripts in numerical order
